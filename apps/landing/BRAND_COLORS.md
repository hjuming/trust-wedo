## Core Colors

| Name | Hex | Usage | Tailwind Class |
|------|-----|-------|----------------|
| **Deep Navy (Primary)** | `#1a2332` | Main branding, Header, Hero background | `bg-brand-navy` |
| **Trust Blue (Secondary)** | `#2563eb` | Primary buttons, links, icons | `bg-brand-blue` |
| **Tech Cyan (Accent)** | `#06b6d4` | Highlights, badges, borders | `bg-brand-cyan` |
| **Success Emerald** | `#10b981` | Success states, trust signals | `text-brand-success` |
| **Neutral Slate** | `#475569` | Body text, descriptions | `text-brand-slate` |
| **Off-White** | `#f8fafc` | Backgrounds, light text | `bg-brand-light` |

## Theme Mapping

### Light Mode
- Background: `brand-light` (#f8fafc)
- Text: `brand-navy` (#1a2332)
- Secondary Text: `brand-slate` (#475569)

### Dark Mode
- Background: `brand-navy` (#1a2332)
- Text: `brand-light` (#f8fafc)
- Secondary Text: `brand-light/70`
